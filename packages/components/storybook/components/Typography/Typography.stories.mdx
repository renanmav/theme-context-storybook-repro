# Typography

## Props

<Props of={Stories.ofProps} />

### variant

Which text variant that will be used to shown the message. Accepts either one of the following:
- `h1`
- `h2`
- `h3`
- `h4`
- `btn1`
- `btn2`
- `btn3`
- `b1`
- `b2`
- `cap`
- `a1`
- `a2`
