## Props

<Props of={Stories.ofProps} />

### size

Which size the button should have. Accepts `large`, `medium` or `small`. Large and Medium buttons have a 7% gutter on both sides.

<Preview withSource="none">
  <Story name="size">
    <Stories.size />
  </Story>
</Preview>

### type

Tells which type this button has. Accepts `primary`, `inverse`, `elevated` or `outline`.

  <Story name="type">
    <Stories.type />
  </Story>
</Preview>

### disabled

Prevent all interactions with the button.

<Preview withSource="none">
  <Story name="disabled">
    <Stories.disabled />
  </Story>
</Preview>

### onPress

Called when the button is pressed by a pointer or keyboard.

  <Story name="onPress">

## Example of usage

```js
<Button text="Press me" type="inverse" size="large" onPress={() => console.log('Hello world')} />
```
